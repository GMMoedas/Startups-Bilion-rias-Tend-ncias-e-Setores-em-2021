startups-bilionarias-tendencias-e-setores-em-2021
Este projeto apresenta uma análise detalhada das empresas que se tornaram unicórnios em 2021 — startups avaliadas em mais de US$ 1 bilhão.   O objetivo é identificar tendências de mercado, setores mais promissores e países com maior concentração de unicórnios, utilizando técnicas de análise de dados e visualização
